# Arkanoid
The classic Arkanoid game. Written in Java.

## Installation
Clone the repository:
```bash
git clone "https://github.com/idokatzav/Arkanoid.git"
```

## Running the game
1. Compile with ant:
'''bash
ant compile
'''
2. Run the game:
Regular order (1->2->3->4):
'''bash
ant run
'''
Costumized order:
'''bash
ant run -Dargs="[level1] [level2] ..."
'''
